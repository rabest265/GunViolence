What factors affect incidence of gun violence?



The chart above examines how factors such as number of gun laws, median income, and population may affect the number of incidents of gun violence per state. Each plot point represents the number of gun violence incidents in a state reported for a year.

There does not appear to be a clear relationship at state level between the number of gun laws versus incidents, nor does the state median income show an obvious effect on number of incidents. However, this dataset shows gun violence incidents clearly increase with state population size. Also, the chart shows the overall number of gun violence incidents increasing from 2014-2017.
